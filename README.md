
## Informacje
Aplikacja WeatherApp jest aplikacja webową umozliwiającą sprawdzanie pogody w danym miejscu.
Tworząc projekt miałem nacelu naukę obsługi plików JSON i operacji na potrzebnych mi informacjach.

## Technologie
* Java
* Spring
* Css
* Html
* Thymelaf
